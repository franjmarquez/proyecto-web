A patch 12 multi-platform hotfix is now available!

 2/3/2022

Hotfix 1.1.12.168 is now available on Xbox, Steam, Epic, and PlayStation! It includes fixes to issues with the B button on Xbox and the game not saving for new players on PC.

Hotfix Notes

  • Scooted some props to prevent players from getting stuck behind them
  • Fix for certain music layers not triggering properly in frightening situations
  • Fix for a missing geometry visuals in a few hard-to-find locations
  • Fix for a rare situation causing a persistent slowdown

Console-Only Changes

  • Fix for the B/cancel button becoming permanently unresponsive after actions in the keybind menu
  • Other input rebinding fixes

Steam and Epic-Only Changes

  • Fix for some cases where the game would not save for new players
